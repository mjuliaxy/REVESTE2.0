BANCO DE DADOS CADASTRO/LOGIN


Como rodar:
Abra o terminal na pasta `reveste-api`.
Instale as dependências:
    ```bash
    npm install
    ```
 Inicie o projeto:
    ```bash
    node index.js
    ```
 Acesse o navegador:
      Site Completo: `http://localhost:8080`
    **Página de Login Direta: `http://localhost:8080/html/login.html`



BANCO DE DADOS VENDEDOR 


abrir o terminal em  svendedor(ou na pasta):

npm init -y
npm install express sqlite cors
node svendedor.js

se der erro:
abre o powershell como admin e rode o comando Set-ExecutionPolicy RemoteSigned 
pra testar se deu certo abre na porta http://localhost:3000/vendedores
